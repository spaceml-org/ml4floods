Install

InstallÂ¶

ML4Floods is python package to do end-to-end flood extent estimation from optical images using deep learning models.

To install the package run:

pip install git+https://github.com/spaceml-org/ml4floods#egg=ml4floods

The package is available in GitHub. This package is developed and maintained in Linux although most of its functions should work in Windows too. The dependencies of the package are listed in the requirements.txt file.

ML4Floods is published under a GNU Lesser GPL v3 licence
