Viewer and label editor¶

  • Author: Gonzalo Mateo-García

Brief description¶

The ml4floods viewer and label editor is a Flask web application to view and manually edit the floodmaps of the WorldFloods dataset. We use this application to keep growing WorldFloods and to improve the quality of its labels. In order to test it, you need to download the Worldfloods dataset (at least a subset of it).

To launch the viewer clone the package and run:

git clone git@github.com:spaceml-org/ml4floods.git
cd ml4floods/viewer

python serve.py --root_location /path/to/worldfloods_v1_0/ --gt_version v1 --no_save_floodmap_bucket
